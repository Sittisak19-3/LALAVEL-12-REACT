# React State Workshop Summary

โปรเจกต์นี้จัดทำขึ้นเพื่อเรียนรู้การใช้งาน **React State** ร่วมกับระบบหลังบ้าน **Laravel 12** และ **Inertia.js**

## เวิร์กชอปที่ทำสำเร็จในบทนี้
1. **Circle and State (`/circle`)** - เรียนรู้การอัปเดต State เบื้องต้นผ่านการสลับสีของวงกลม
2. **Counter App (`/counter`)** - แอปนับเลขเพื่อฝึกคำนวณและเปลี่ยนแปลงสถานะตัวเลข (เพิ่ม/ลด/รีเซ็ต)
3. **Form Example (`/form-example`)** - การทำ Data Binding ผูกข้อมูลจาก Input เข้ากับ State แบบ Real-time
4. **List Manager (`/list-manager`)** - จัดการข้อมูลแบบ Array State สามารถเพิ่ม-ลบการ์ดรูปภาพที่ดึงมาจาก Picsum Cloud ได้
5. **Volume Toggle (`/`)** - ระบบประยุกต์ใช้ปุ่มกดเปิด-ปิดเสียง สลับสถานะตัวแปร Boolean พร้อมเปลี่ยนไอคอนและสีปุ่ม

## เทคโนโลยีที่ใช้
- Laravel 12
- React (JSX)
- Inertia.js
- Git / GitHub
